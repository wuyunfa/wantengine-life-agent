# WantEngine 生命化智能体

面向“从工具到生命”的开源智能体原型工程。

## 项目定位
本项目尝试将智能体从“外部任务驱动”推进到“内在生命逻辑驱动”，围绕七个核心方向构建可运行原型：

- 匮乏驱动（Want Engine / Vitalis）
- 有限性契约（Finis）
- 内在世界层（私有记忆边界）
- 非理性决策注入（受控偏离）
- 意义自赋（动态使命）
- 依恋与排他协议
- 群体遗传与变异（Eidolon）

## 快速开始
```bash
python -m pip install pyyaml
python sim/run_demo.py
```

## 当前能力
- 五维匮乏状态衰减与压力驱动行为
- 生命周期信用模型
- 依恋优先级请求路由
- 基因融合与突变示例
- 桌宠实验脚本（可选）
